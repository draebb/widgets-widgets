Improve widget management.

== Description ==

Widgets Widgets improves widget management with the following features:

* Remembers the open/close state of the widget areas.
* Makes the widgets in the Available Widgets area hide-able.

*IE7 is not supported.*

== Installation ==

1. Unzip the archive and put the `widgets-widgets` folder into the `/wp-content/plugins/` folder.
1. Activate the plugin through the 'Plugins' menu in WordPress.
